# Dar-Al-Hrf
A web-based e-commerce platform dedicated to celebrating and showcasing the artistry of the Year of Handicrafts 2025, inspired by the Ministry of Culture's initiative. This project provides a digital marketplace for artisans to display their traditional crafts and for users to explore and purchase these unique cultural pieces.
# 🚀 Project Overview

This website serves as a bridge between traditional craftsmanship and modern digital accessibility. It is designed to preserve cultural heritage while providing a seamless shopping experience for enthusiasts of authentic handicrafts.
# Key Features

    Artisan Showcase: Dedicated pages to view handicrafts by category, region, or artist.

    Secure Browsing: Intuitive catalog navigation with product details.

    Admin Dashboard: Backend interface for managing inventory, categories, and customer orders.

# 🛠️ Technical Stack

This project is built using a classic LAMP-style architecture:

    Frontend: HTML5, CSS3, and JavaScript (Vanilla)

    Backend: PHP 8+

    Database: MySQL (via phpMyAdmin)

    Server: Local development environment (e.g., XAMPP, WAMP, or MAMP)
